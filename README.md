# MiMo Staking Calculator

Multi-protocol staking rewards calculator. Built with **MiMo V2.5**.

## Features

- Multi-protocol: Lido (ETH), Cosmos (ATOM), Marinade (SOL), Polkadot (DOT)
- Compounding vs simple interest calculation
- Break-even analysis for gas costs
- Multi-year projections
- Pool comparison

## Usage

```bash
python calculator.py --principal 10000 --days 365
python calculator.py --compare
python calculator.py --projection 5
python calculator.py --break-even 0.005
```
